About this project

Loan Eligibility & Advisory System

This web application is a single, modern interface that demonstrates how banks and financial institutions think about loan applications — enriched with explainable AI and clear visualisation. It is built on top of a trained loan prediction model and exposes the main decision drivers in a way that applicants can actually understand.

What the system calculates

  • • Loan eligibility (Yes / No) using the trained model.
  • • Recommended loan type based on income, loan size, assets, education, and CIBIL.
  • • EMI using the standard banking formula, plus DTI, LTI, and Applicant Affordability Index.
  • • Maximum safe loan amount and a risk category from Very Safe to Very Risky.
  • • Multi‑bank EMI comparison at fixed benchmark rates for quick decisioning.

Explainable AI & SHAP

Under the hood, the model uses SHAP (SHapley Additive exPlanations) to reveal which input features most strongly influenced the decision. Instead of showing a full technical plot, we surface a concise list of the top three drivers — clearly marking which ones supported approval and which ones contributed to rejection.

This bridges the gap between complex machine learning models and the transparent, auditable explanations that regulators and customers increasingly expect.

How to use this tool effectively

  • 1. Go to the Loan Analyzer page.
  • 2. Either enter your details manually, or use the Generate Profile button to simulate realistic scenarios.
  • 3. Review the eligibility outcome, affordability metrics, risk category, and SHAP explanations.
  • 4. Experiment with different combinations to see how income, CIBIL, and assets impact approval.

Note: This prototype is meant for academic, demonstration, or early advisory use. Final credit decisions always depend on the detailed policies and documentation checks of each lending institution.
